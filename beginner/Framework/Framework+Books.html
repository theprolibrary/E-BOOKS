FRAMEWORK

A framework, or software framework, is a platform for developing software applications. It provides a foundation on which software developers can build programs for a specific platform. A framework is similar to an application programming interface (API), though technically a framework includes an API. A framework may also include code libraries, a compiler, and other programs used in the software development process.
